# ACLGraph Operator Optimization on Ascend NPU

## Objective

This project focuses on **ACLGraph operator optimization** for Ascend NPU, aiming to improve the performance of vLLM on Huawei Ascend hardware.

## Scope

- Identify performance bottlenecks in current ACLGraph operator implementations
- Optimize kernel performance for key operators used in LLM inference
- Benchmark and validate improvements against baseline
